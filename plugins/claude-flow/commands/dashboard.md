## Data Collection

### 1. Apps Status

```bash
# List all apps
ls -d apps/*/ | grep -v devops

# For each app, check:
# - Current branch
git -C apps/[name] branch --show-current 2>/dev/null || git branch --show-current

# - Has Dockerfile
test -f apps/[name]/Dockerfile && echo "yes" || echo "no"

# - Build status (check last build)
test -f apps/[name]/dist && echo "built" || echo "not built"

# - Test status
cd apps/[name] && npm test --passWithNoTests 2>&1 | tail -5

# - Lint status
cd apps/[name] && npm run lint 2>&1 | tail -5
```

### 2. Environment Status

```bash
# Local Docker status
cd apps/devops/docker && docker-compose ps

# Check service health
docker-compose ps --format json | jq '.[] | {name: .Name, state: .State}'
```

### 3. Sprint Status

```bash
# Find current sprint
current_sprint=$(ls -t project/sprints/SPRINT-*.md 2>/dev/null | head -1)

# Parse sprint file for status
grep -E "^Status:|^Progress:|^Stories:" "$current_sprint"

# Count stories by status
grep -l "Status: Done" project/backlog/**/*.md | wc -l
grep -l "Status: In Progress" project/backlog/**/*.md | wc -l
grep -l "Status: Review" project/backlog/**/*.md | wc -l
```

### 4. Current Work

```bash
# From session.json
cat .claude/session.json | jq '.current_story, .current_branch'

# Git status
git status --short
git diff --stat
```

### 5. Pull Requests

```bash
# List open PRs
gh pr list --state open --json number,title,reviewDecision,createdAt

# List recently merged
gh pr list --state merged --limit 5 --json number,title,mergedAt
```

### 6. Health Checks

```bash
# Technical debt count
ls project/backlog/technical/TD-*.md 2>/dev/null | wc -l

# Test coverage (if available)
cat apps/[name]/coverage/coverage-summary.json | jq '.total.lines.pct'

# Outdated dependencies
cd apps/[name] && npm outdated --json | jq 'length'

# Vulnerable dependencies
cd apps/[name] && npm audit --json | jq '.metadata.vulnerabilities.total'
```
